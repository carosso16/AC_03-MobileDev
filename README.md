# App de Controle de Gastos

Este projeto é um aplicativo simples de controle de gastos pessoais criado com Expo e React Native. Ele permite:

- registrar gastos com descrição e valor
- exibir todos os gastos em uma lista
- ver o total acumulado automaticamente
- apagar despesas registradas
- navegar entre a lista e o formulário de cadastro

## Como usar

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Inicie o app:

   ```bash
   npx expo start
   ```

3. Abra no Expo Go ou em um simulador.
